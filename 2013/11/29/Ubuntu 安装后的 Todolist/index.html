Ubuntu 安装后的 Todolist

发表于 2013-11-29   |  

Ubuntu 12.04 64-bit 桌面版简单配置，以下为 todolist

摘要

安装 Ubuntu 后的 todolist，关键点：

  1. 修改软件源，用 aptitude 统一管理软件
  2. 按需设置代理
  3. 配置自己的环境变量、别名

安装虚拟机工具

以 VMware 为例：

  1. 点击 【虚拟机】->【安装VMware工具】
  2. 将压缩包拖至 ~/download，后用 tar 解压
  3. 运行 vmware-install.pl，以默认配置即可（全回车）

设置 root 密码

修改软件源

打开 Ubuntu 软件中心，并点击 【编辑】->【软件源】

  1. 设置常规软件源
    从 下载自 中选择一个中国的服务器即可

  2. 添加额外软件源
    点击 【其它软件】->【添加】
    通常添加 deb 或 ppa：
    如需 chrome，输入 deb http://dl.google.com/linux/chrome/deb/ stable main
    如需 oracle jdk，输入 ppa:webupd8team/java
    see more[https://launchpad.net/~webupd8team/+archive/ubuntu/java]
    如需 vim7.4，输入 ppa:fcwu-tw/ppa

所做的修改其实均在 /etc/apt 下：
修改的服务器源在 /etc/apt/sources.list 文件中
添加的软件源在 /etc/apt/sources.list.d 文件夹里

添加 ppa 也可以如下方式：

设置中文

打开系统设置，并点击 【语言支持】
从 添加或删除语言 中选择 简体中文 ，后将其拖至列表首，并点击 应用至整个系统 即可
依照提示安装需要的语言包

安装 aptitude

与 apt-get 相比，aptitude 命令统一，更友好

设置代理

VPN 代理

在右上方点开网络设置，点击 【VPN连接】->【配置VPN】
添加并编辑 VPN
最后设置 【VPN】->【高级】->【使用点到点加密】（很重要）

以上完成后，可选择升级已有软件
