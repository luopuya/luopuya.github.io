Linux 压缩中文文件名会乱码，这让需要在多平台上学习、工作的人难以满意。

不过解决的方案还是有的，请看下文

摘要

Linux 压缩中文文件名关键点：使用别的压缩工具= =

解决

Linux 的 tar 并不保存编码集，有如下解决方式：

1. 使用 7zip 或 rar

7zip 安装如无权限，修改 install.sh 里的DEST_HOME

注：可参考笔者上篇博文

2. 人工/脚本转换文件名

文件内容并不会乱码，因而临时人工修改，或用脚本做映射也是权宜之计
